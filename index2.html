Как я сделал данный сайт

Заходим на сайт нажимаем на вкладку Examples, ниже будет надпись Starters, выбираем CDN starter, после этого вас перекинет на github , в самом низу сайта будет три строчки которые нужно будет ввести в терминал:
git clone https://github.com/twbs/examples.git
cd examples/starter/
open index.html

После того как вы выполнили шаг #1 у вас появился файл index.html, заходим в него через редактор кода(Visual Studio Code), и вот вы уже можете верстать свой сайт, делать весь его дизайн, так же есть css ,но он отвечает за цвета,шрифты,стили,и т.д.

После того как вы выполнили все шаги вы можете загрузить ваш сайт на github,
Cоздаете свой аккаунт, нажимаем на кнопку новый репозиторий даем ему имя и в конце пишем .github.io
Создайте новый файл. Когда репозиторий не пуст, то в верхнем правом углу есть кнопка "Create New File",
Введите имя index.html и скопируйте туда содержание файла index.html, который у вас получился в предыдущем шаге,
Нажмите кнопку "Commit new file",Повторите то же самое для файла style.css,Подтвердите публикацию на
GitHub Pages. В настройках репозитория: Settings -> Раздел Options -> Секция GitHub Pages.
Через несколько мгновений страница станет доступна по вашему адресу https://Ваше имя.github.io
